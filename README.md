### Часть 1: Код программы (main.py)

```python
import string

def analyze_text():
    print("--- Программа анализа текста ---")
    
    # 1. Принимаем текст от пользователя
    user_input = input("Введите предложение или абзац для анализа: \n")
    
    if not user_input.strip():
        print("Ошибка: текст не может быть пустым.")
        return

    # Предварительная очистка: приводим к нижнему регистру и удаляем знаки препинания
    clean_text = user_input.translate(str.maketrans('', '', string.punctuation)).lower()

    # 2. Разбиваем текст на слова
    words = clean_text.split()
    
    # 3. Подсчитываем общее количество слов
    total_words = len(words)
    
    # 4. Находим повторяющиеся слова (и частоту их появления)
    word_freq = {}
    for word in words:
        word_freq[word] = word_freq.get(word, 0) + 1
    
    # Фильтруем только те, что повторяются (больше 1 раза)
    repeating_words = {word: count for word, count in word_freq.items() if count > 1}
    
    # Вывод результатов
    print("\n--- Результаты анализа ---")
    print(f"Общее количество слов: {total_words}")
    
    if repeating_words:
        print("Повторяющиеся слова и их количество:")
        # Сортируем по убыванию частоты
        sorted_repeating = sorted(repeating_words.items(), key=lambda x: x[1], reverse=True)
        for word, count in sorted_repeating:
            print(f" - '{word}': {count} раз(а)")
    else:
        print("Повторяющихся слов не найдено.")

if __name__ == "__main__":
    analyze_text()
```

# Задание №4: Анализ текста на Python

## Цель проекта
Изучение методов работы со строками (`string`) и списками (`list`), а также освоение базовых алгоритмов обработки текстовых данных.

## Задачи
1. Реализовать ввод текста пользователем.
2. Очистить текст от знаков препинания и привести к единому регистру.
3. Разбить текст на отдельные слова.
4. Вычислить общее количество слов.
5. Найти и вывести слова, которые встречаются в тексте более одного раза.

## Реализация
В программе использованы:
* Метод `.split()` для токенизации строки.
* Метод `.translate()` со словарем из `string.punctuation` для эффективного удаления знаков препинания.
* Словарь (`dict`) для подсчета частоты слов (алгоритм с временной сложностью $O(n)$).

## Ответы на контрольные вопросы
1. **Как разбить строку на список слов?** Использовать метод `str.split()`. По умолчанию он разбивает строку по пробельным символам.
2. **Как подсчитать количество элементов?** С помощью встроенной функции `len(имя_списка)`.
3. **Как найти повторяющиеся значения?** Наиболее эффективный способ — использовать словарь, где ключом является элемент, а значением — количество его вхождений в список.
